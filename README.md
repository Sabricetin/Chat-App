Chat-App: Gerçek Zamanlı Mesajlaşma Uygulaması

Bu proje, Swift ile iOS cihazlar için geliştirilen bir gerçek zamanlı sohbet uygulamasıdır. Firebase servislerini kullanarak kullanıcı kimlik doğrulama, veritabanı yönetimi ve gerçek zamanlı mesajlaşma özelliklerini destekler.

Özellikler 🚀
	•	Kullanıcı Kimlik Doğrulama: Firebase Authentication ile kullanıcı yönetimi.
	•	Gerçek Zamanlı Mesajlaşma: Firebase Realtime Database kullanılarak anlık mesaj gönderme ve alma.
	•	Mesaj Kaydı: Mesajlar Firebase üzerinde dinamik olarak depolanır ve çağrılır.
	•	Okunmamış Mesaj Takibi: Mesajların alıcı tarafından okunup okunmadığını izler.
	•	Klavye Yönetimi: Klavye açıldığında veya kapandığında UI otomatik olarak ayarlanır.
	•	Duyarlı Arayüz: Kullanıcı deneyimini artırmak için düzgün ve duyarlı bir tasarım.

Kurulum ve Kullanım 📲
	.	Projeyi Klonlayın:

git clone https://github.com/your-username/chat-app.git
cd chat-app


	1.	Bağımlılıkları Yükleyin:
	2•	Firebase için gerekli pod’ları yükleyin:

pod install


	1.	Firebase Yapılandırması:
	2•	Firebase Console üzerinden bir proje oluşturun.
	3•	GoogleService-Info.plist dosyasını indirin ve proje dosyasına ekleyin.
	4.	Projeyi Çalıştırın:
	5•	Xcode ile projeyi açın ve cihazınızda veya simülatörde çalıştırın.

Kullanılan Teknolojiler 🛠️
	•	Swift: iOS uygulama geliştirme.
	•	Firebase Realtime Database: Gerçek zamanlı veri senkronizasyonu.
	•	Firebase Authentication: Kullanıcı kimlik doğrulama.
	•	UIKit: Kullanıcı arayüzü geliştirme.
	•	Auto Layout: Duyarlı ve esnek UI tasarımı.
